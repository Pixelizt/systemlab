# See the full Claude Handoff Prompt

The complete handoff prompt for Claude AI is available in the artifact directory.
To use it, copy the full contents of `claude_handoff_prompt.md` from the Antigravity conversation artifacts.

Alternatively, Claude should read these files in order:
1. `SYSTEMLAB_MASTER_HANDOFF.md` (this repo)
2. `chat_history/` (all files, oldest first)
3. `DESIGN.md` (design system)
4. `LEADLAB_MASTER_HANDOFF.md` (in the leadlab-crm repo)
5. `chat_history/` (in the leadlab-crm repo, all files)
